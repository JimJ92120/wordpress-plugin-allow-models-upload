# wordpress-plugin-allow-models-upload

Allow users to upload <em>.gltf</em> and <em>.glb</em> files into WordPress Media Library.

---
## Requirements

### For plugin installation
|             |          |
|-------------|----------|
| `PHP`       | `>= 7.4` |
| `WordPress` | `>= 5.9` |

### For development
|             |           |
|-------------|-----------|
| `PHP`       | `>= 7.4`  |
| `WordPress` | `>= 5.9`  |
| `npm`       | `>= 6.0`  |
| `node`      | `>= 14.0` |
